### Housing Price Prediction Project


### Objective - The objective of this project is to predict housing prices using machine learning techniques based on various socio-economic and geographical features. The project demonstrates the complete ML workflow from data preprocessing to model deployment readiness.


### Project Structure - 
## Dataset
- Source: California Housing Dataset
- Type: Tabular data
- Target Variable: Median House Value
- Features include:
  - Median Income
  - House Age
  - Average Rooms
  - Population
  - Latitude & Longitude

## Technologies Used
- Python
- Jupyter Notebook
- NumPy
- Pandas
- Matplotlib / Seaborn
- Scikit-learn
- Git & GitHub

## Machine Learning Models
- Linear Regression
- Multiple Linear Regression
- Random Forest Regressor (Best Performing Model)

The final trained model is saved as: 'best_rf_model.pkl'


### Workflow - 
1. Data Loading
2. Data Preprocessing
   - Missing value handling
   - Feature scaling
3. Exploratory Data Analysis (EDA)
4. Model Training
5. Model Evaluation
6. Model Saving
